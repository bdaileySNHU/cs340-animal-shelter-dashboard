# cs340-animal-shelter-dashboard
An interactive dashboard built using MongoDB, Python, and Dash.

# Reflection
This project refects how to focus on the client’s needs and translate them into a functional and user-friendly solution. This includes mapping the specific search and rescue dog requirements to the MongoDB queries and incorporating them into the Dashboard. I utilized Object-Oriented Programming principles to create an independent and reusable Python CRUD module. Abstracting the database connections and query logic away from the Dash front-end. This makes the code more readable and maintainable.
